# Tomate

Um pomodoro no CLI.

# Uso

```sh
# 4 x 25/5
tomate sprint

# Repete 15/3 indefinitamente
tomate custom 15 3
```

1. Para pausar o relógio, aperte `ENTER`.
2. Para pular a contagem atual, escreva `p` e aperte `ENTER`.

# Instalação

Após clonar o repositório, entre na pasta e rode:
```sh
 cargo install --path .
```
